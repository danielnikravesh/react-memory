# react-memory-simple

```javascript
    git clone https://github.com/LarsBergqvist/react-memory-simple
    cd react-memory-simple
    yarn install
    yarn start
```
<p>Browse to http://localhost:3000
<p>Click the cards to flip to images and try to find image pairs<p>
